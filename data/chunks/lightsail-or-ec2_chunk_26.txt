Amazon EC2 instance types
When you launch an EC2 instance, the instance type that you specify determines the hardware of the host computer used for your
instance. Each instance type offers different compute, memory, and storage capabilities, and is grouped in an instance family based
on these capabilities. This guide walks you through EC2 instance types.
