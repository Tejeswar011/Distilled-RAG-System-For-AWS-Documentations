only
applies to buckets that have public policies. However, if you add a public
statement to the policy,
RestrictPublicBuckets
takes effect on the
bucket. It allows only AWS service principals and authorized users of the
bucket owner's account to access the bucket.

A statement that grants access to AWS CloudTrail (which is an AWS service
principal)

A statement that grants access to the public, for example by
specifying
"Principal": "*"
with no limiting
Condition
